# Deep Research Agent

## 프로젝트 개요
AutoGen을 활용한 심층 연구 에이전트 프로젝트입니다.

## 기술 스택
- **Python**: uv를 사용한 패키지 관리
- **AutoGen**: AI 에이전트 프레임워크
- **OpenAI API**: LLM 모델
- **Firecrawl**: 웹 크롤링 도구

## 환경 설정
- `.env` 파일에 다음 환경 변수 필요:
  - `OPENAI_API_KEY`: OpenAI API 키
  - `FIRECRAWL_API_KEY`: Firecrawl API 키

## 개발 가이드
- Python 3.13+ 사용
- uv를 통한 의존성 관리
- AutoGen의 베스트 프랙티스 준수

## 프로젝트 목표
웹에서 정보를 수집하고 분석하여 심층 연구를 수행하는 자동화된 에이전트 시스템 구축
